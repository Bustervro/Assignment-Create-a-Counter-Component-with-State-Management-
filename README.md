# React State Counter App

This is a simple React app built with Vite. It demonstrates React state management using the useState Hook.

## Features

- Increment count by 1
- Increment count after a 2-second delay
- Demonstrate React batching with Increment Twice
- Correctly increment twice using updater functions
- Reset count back to 0

## How to Run

```bash
npm install
npm run dev
